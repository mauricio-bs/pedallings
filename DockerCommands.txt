Criar uma imagem a partir do dockerfile
$ docker build -t nome/tag(dockernode) .


Cria o container
$ docker run -p 3000:3000 -d tgmarinho/dockernode

-d: detach, ou seja, ao terminar fica livre e o processo roda em background, porém exibe o ID do container


Ver os logs do que está acontecendo no container
$ docker logs ID

docker-compose: É um orquestrador de containers, definindo o comportamento de cada container na aplicação

Cria o container e sobe o serviço do docker-compose, o qual faz o build do projeto conforme o dockerfile
$ docker-compose up

Roda o comando em background para liberar o terminal
$ docker-compose up -d
